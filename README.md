## 🚀 Tính Năng

- Hiển thị danh sách sản phẩm công nghệ (laptop, điện thoại, phụ kiện...)
- Gọi API để lấy dữ liệu sản phẩm

## 🔧 Công Nghệ Sử Dụng

- **Back-end**: Laravel + API 
- **Mobile**: Flutter
